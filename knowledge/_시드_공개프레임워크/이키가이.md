# 이키가이 (Ikigai) — 삶의 방향 찾기

> 공개 커리어 프레임워크. 출처 표기 시: `📎 근거: 이키가이 (섹션 N)`.

## 섹션 1. 개념
이키가이(生き甲斐)는 "살아가는 보람"을 뜻하는 일본어예요. 커리어에서는 **네 가지 원이 겹치는 지점**에서 지속가능한 일을 찾자는 프레임으로 널리 쓰입니다.

## 섹션 2. 네 가지 원
- **좋아하는 것 (What you love)** — 시간 가는 줄 모르고 하는 일
- **잘하는 것 (What you're good at)** — 남보다 수월하게 해내는 강점
- **세상이 필요로 하는 것 (What the world needs)** — 수요·사회적 쓸모
- **보상받는 것 (What you can be paid for)** — 시장이 돈을 지불하는 일

## 섹션 3. 겹침 영역
- 좋아함 + 잘함 = **열정(Passion)**
- 잘함 + 보상 = **직업(Profession)**
- 보상 + 세상필요 = **소명(Vocation)**
- 세상필요 + 좋아함 = **사명(Mission)**
- 네 개가 다 겹치는 한가운데 = **이키가이**

## 섹션 4. 코칭 적용 질문
- "지금 일에서 네 원 중 어디가 채워지고, 어디가 비어 있나요?"
- "비어 있는 원을 채우려면 지금 자리에서 바꿀 수 있는 게 있을까요, 자리를 옮겨야 할까요?"
- 주의: 이키가이는 '완벽한 하나의 천직'을 찾으라는 게 아니라, 겹침을 **넓혀가는 방향**을 잡는 도구로 쓰는 게 좋아요.
