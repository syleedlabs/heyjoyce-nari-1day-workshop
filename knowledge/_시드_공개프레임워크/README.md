# _시드_공개프레임워크 — 빌려온 예제 지식

이 폴더는 워크숍 **조기 성공용 예제 지식(시드팩)**입니다. 나리님이 아직 본인 자료를 넣기 전에, 코치가 실제로 어떻게 근거를 찾아 답하는지 **먼저 성공을 맛보게** 하는 용도예요.

## 이 폴더의 자료는

- **공개된 커리어 프레임워크**(이키가이·GROW·커리어 앵커)를 정리한 것입니다. 특정 개인의 상담 정보가 아니에요.
- 그래서 **공개 레포에 올라가도 안전**합니다. (개인 자료가 담기는 `knowledge/원본/`은 `.gitignore`로 차단되지만, 이 시드 폴더는 공개 자료라 커밋됩니다.)
- **나리님의 지식이 아니라 "빌려온 지식"**이에요. Part 2에서 본인 자료로 인제스트를 시작하면, 코치의 진짜 심장은 `knowledge/원본/`·`_인덱스.md`로 옮겨갑니다.

## 어떻게 쓰이나

- **2-0 (Part 2 워밍업)** — 이 시드로 `/2-0-seed-answer`를 돌려 코치의 첫 그라운딩 답변(📎 근거 표기)을 받습니다. **보장된 첫 성공.**
- **2-1 이후** — 본인 자료를 넣기 시작하면 코치는 본인 자료를 우선 참조합니다. 시드는 참고용 배경지식으로 남아요.

## 시드를 지우고 싶다면

본인 자료가 충분히 쌓이면 이 폴더를 통째로 지워도 됩니다. 코치에게 "시드 프레임워크 폴더를 빼줘"라고 하면 정리해줘요. (남겨두면 일반 커리어 이론 배경으로 계속 쓰입니다.)
