# knowledge/ — 코치의 지식 베이스

이 폴더가 커리어 코치 에이전트의 **심장**입니다. 코치는 여기 있는 자료에만 근거해 답합니다.

## 구조

```
knowledge/
├── README.md          ← 이 파일
├── _인덱스.md          ← 구조화된 지식 인덱스 (2-2에서 작성)
└── 원본/               ← 인제스트한 자료 (2-1에서 채움)
    ├── <강의명>.md      ← PPT에서 뽑은 텍스트
    ├── <영상명>.md      ← 영상 자막/전사
    └── <주제>_상담노트.md ← 익명화한 상담 경험
```

## 원칙 (중요)

1. **익명화 필수** — 상담 노트의 내담자 실명·회사·개인정보는 인제스트 전에 가명/일반화. (부록 익명화 가이드 참고)
2. **로컬 보관** — 이 폴더의 실자료는 **공개 레포에 올리지 않습니다.** `.gitignore`에 `knowledge/원본/`이 등록돼 있어요. 워크북·구조만 공개, 지식은 본인 컴퓨터에만.
3. **출처 표기의 근거** — 코치가 "📎 근거: <자료명>"을 붙일 때 그 `<자료명>`이 이 폴더의 파일명입니다. 파일명을 알아보기 쉽게 지어주세요.

## 워크숍 진행

- **2-1** 인제스트: PPT·영상·노트 → `원본/`
- **2-2** 구조화: 프레임워크·핵심개념·자주 나오는 질문을 `_인덱스.md`로
- **2-3** 첫 그라운딩 답변: 코치가 이 자료를 인용해 답
