# 1-3. 코치 작명 + 첫 만남

::: tip Step 3 / 13 · Part 1
이 챕터에서 — **코치에게 이름을 지어주고**, 코치의 첫 인사를 듣고, 코치가 나리님님을 인터뷰해 페르소나 재료를 모읍니다.
:::

## [실습] 작명 (1분)

코치에게 이름을 지어주세요. **딱 1분**입니다. 길게 고민하지 마세요 — 마음에 안 들면 나중에 언제든 바꿀 수 있어요.

작명 가이드:
- **짧게 (2~4자)** — 매일 부를 이름이라 호흡이 길면 부담
- **부르기 좋게** — 발음이 자연스러운지
- **본인 취향 반영 OK** — 영문/한글, 코치·멘토·동반자 컨셉, 좋아하는 어감

혼자 정하기 막막하시면 코치가 후보를 제안하게 할 수 있어요. 클로드 코드 터미널에서 입력해 주세요.

```
/1-3-name-helper
```

**기대 결과** — 나리님님의 코칭 도메인·톤 취향에 맞춘 이름 후보 3개. 마음에 드는 걸 고르시거나, 본인 아이디어로 정하셔도 됩니다.

이름이 정해지면 코치의 페르소나 파일(`.claude/agents/코치.md`)이 그 이름으로 이름표를 답니다. 이 파일은 다음 챕터에서 본격적으로 채워져요.

## [실습] 코치의 첫 한 마디 ⭐

::: tip 첫 만남
코치가 살아 움직이기 시작하는 순간입니다. 클로드 코드 터미널에서 입력해 주세요.

```
/1-3-first-hello
```
:::

**기대 결과** — 코치가 나리님님 이름을 부르며, 자기가 세 가지 모드로 무엇을 할 수 있는지 첫 인사를 건넵니다.

> "나리님님, 안녕하세요. 저는 **코치**입니다.
>
> 오늘부터 저는 나리님님의 지식을 학습한 커리어 코치 분신이 됩니다. 세 가지로 함께할게요 — 나리님님과 **토론**하고(`/spar`), 나리님님 대신 **리서치**하고(`/research`), 코치이 질문에 나리님님이라면 할 **답**을 대신 합니다(`/answer`).
>
> 아직 제 안은 비어 있어요. 나리님님의 자료로 저를 채우는 게 오늘 할 일입니다. 잘 부탁드려요."

이 한 줄이 코치의 첫 발자국이에요. 아직 나리님님 지식을 학습하기 전이라 코치는 "빈 그릇" 상태지만, 이름을 갖고 나리님님을 부르는 순간 애착이 생깁니다.

마음에 안 드는 부분이 있다면 — 톤이 너무 격식 있다거나 호칭이 어색하다거나 — 그대로 코치에게 말씀해 보세요. **클로드 코드는 대화로 수정**합니다.

## [실습] 코치 인터뷰 — 나를 알려주기

이제 반대로, 코치가 나리님님을 인터뷰합니다. 나리님님이 **어떤 코치인지**를 코치가 알아야 나리님님처럼 답할 수 있으니까요. 클로드 코드 터미널에서 입력해 주세요.

```
/1-3-domain-interview
```

코치가 세 가지를 차례로 묻습니다. 짧게, 본인 표현 그대로 답하시면 돼요. 사전에 준비해 오실 건 없습니다.

| # | 코치가 묻는 것 | 무엇을 답하나 |
|---|---|---|
| 1 | **코칭 도메인** | 어떤 커리어 고민을 주로 다루시는지 · 주 대상 (예: "직무 전환·이직 전략, 3~10년차 직장인") |
| 2 | **화법·톤** | 나리님님의 실제 코칭 말투 (따뜻·경청 / 직설·명료 / 질문형 중 하나, 또는 본인 스타일 한 줄) |
| 3 | **고유 관점 2~3개** | 나리님님이 반복해서 강조하는 커리어 신념·프레임워크 (예: "커리어는 사다리가 아니라 포트폴리오다") |

가장 중요한 건 **2번과 3번**이에요. **화법**은 코치가 나리님님 목소리로 들리게 하고, **고유 관점**은 코치가 나리님님답게 판단하게 만듭니다. 이 둘이 "분신"의 핵심입니다.

답변이 막연하면 코치가 예시를 한 번 더 보여줍니다. 지금 부족해도 괜찮아요 — Part 2에서 나리님님 자료를 넣으면 더 풍부해지고, 대화로 계속 다듬을 수 있습니다.

답변이 끝나면 코치가 정리한 표를 보여주고 확인받아요. 다음 챕터(1-4)에서 이 답변이 페르소나 파일에 자동으로 반영됩니다.

::: tip [체크포인트] 여기까지 하셨다면
- 코치에게 **이름**이 생겼다 (`/1-3-name-helper`)
- 코치가 내 이름을 부르며 **3모드로 첫 인사**를 했다 (`/1-3-first-hello`)
- 코치에게 내 **도메인·화법·고유 관점**을 알려줬다 (`/1-3-domain-interview`)

아직 코치 안은 "빈 그릇"이에요. 다음 챕터에서 이 인터뷰 답변을 코치의 매뉴얼(페르소나)로 굳힙니다.
:::
