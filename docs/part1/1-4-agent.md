# 1-4. 자동화 맛보기 : 코치 만들기

::: tip Step 4 / 9 · Part 1
이 챕터에서 — 강사 에이전트를 봤으니, 이제 나리님의 **커리어 코치를 직접 한 번** 돌려봅니다. Part 2에서 제대로 만들 코치의 **미니 버전**이에요. 오늘의 첫 성공이에요.
:::

## [개념] 미니 코치로 먼저 맛본다

1-3에서 강사의 콘텐츠 에이전트가 리서치→제안→제작을 자동으로 해내는 걸 봤어요. 이제 같은 원리로 움직이는 **나리님의 커리어 코치**를 만들 차례예요.

다만 지금은 **미니 버전**이에요. Part 2에서 이름 짓고, 성격 넣고, 나리님의 진짜 강의·상담을 학습시켜 제대로 만들 거예요. 이 챕터에서는 그 **축소판**을 한 번 돌려, "아, 이렇게 작동하는구나"를 몸으로 익힙니다.

::: tip 왜 미니로 먼저?
바로 본인 자료를 넣어도 되지만, **작동하는 코치를 딱 한 번 먼저 경험**하고 가면 Part 2의 작업이 훨씬 수월해요. 목적지를 눈으로 보고 출발하는 거예요. (다른 워크숍에서 예제 데이터로 결과물을 먼저 뽑아보는 것과 같은 장치예요.)
:::

## [개념] 미니 코치가 아는 것 — 커리어 코칭 도메인

이 미니 코치가 다루는 영역은 **커리어 코칭**이에요 — 이직·직무 전환·커리어 설계 같은 고민이요. 그런데 코치가 아무 말이나 지어내지 않으려면, **근거로 삼을 지식**이 있어야 해요.

아직 나리님 자료를 넣기 전이라, 이 워크북엔 커리어 코칭에서 널리 쓰는 **공개 프레임워크 3개**가 시드(예제 지식)로 들어 있어요. 미니 코치는 이걸 근거로 답합니다.

| 프레임워크 | 무엇을 보는 도구 | 잘 맞는 질문 |
|---|---|---|
| **이키가이** | 좋아함·잘함·세상필요·보상 4가지가 겹치는 지점으로 방향 찾기 | "지금 일이 안 맞는 것 같아요" |
| **GROW 모델** | 목표→현실→선택지→실행의지 순서로 대화를 굴리기 | "뭐부터 해야 할지 막막해요" |
| **커리어 앵커** | "절대 포기 못 하는 것" 8가지로 갈림길 결정을 진단 | "이직할지 말지 모르겠어요" |

미니 코치는 질문을 받으면 이 셋 중 맞는 프레임을 골라, **그 근거를 📎로 밝히며** 답해요. (이 자료는 `knowledge/_시드_공개프레임워크/` 폴더에 들어 있어요.)

::: warning 이건 아직 '빌린' 도메인이에요
이 프레임워크들은 **일반 공개 이론**이라, 코치가 인용할 때 "일반 프레임"으로 표시돼요 — 나리님만의 관점·화법·사례는 아직 없죠. Part 2에서 나리님의 실제 강의·상담을 넣으면, 도메인이 **나리님 본인의 것**으로 바뀝니다.
:::

## [실습] 미니 코치 첫 실행

위에서 본 **시드 프레임워크**로 미니 코치가 실제로 어떻게 답하는지 봅니다. 클로드 코드 터미널에 입력하세요.

```
/1-4-seed-answer
```

미니 코치가 커리어 질문을 하나 물어봐요(못 떠올리면 샘플을 제안해요). 그러면 시드에서 근거를 찾아 이렇게 답합니다.

```
[코치] 답변

<프레임을 적용한 답>

📎 근거: 커리어 앵커 (섹션 3)

💬 이어서 묻고 싶은 것: <다음 질문 1개>

지금은 빌려온 공개 프레임워크로 답했어요. Part 2에서 본인 자료를 넣으면,
같은 질문에 본인 관점과 화법으로 답하게 됩니다.
```

**딱 하나만** 눈여겨보세요 — `📎 근거`가 붙어 있다는 것. 1-3에서 강사 에이전트가 출처를 명기하던 것과 똑같아요. 미니 코치도 **아무 말이나 지어내지 않고, 어디서 나온 말인지 밝히며** 답합니다.

::: warning 아직 완벽하지 않아도 괜찮아요
시드는 일반 이론이라 답이 "교과서처럼" 나올 수 있어요. 정상이에요. 나리님만의 표현·사례·관점은 Part 2에서 본인 자료를 넣을 때 입혀집니다. 지금은 **작동 방식**을 눈에 익히는 게 목적이에요.
:::

## [체크포인트] 첫 성공을 확인해요

- **미니 코치에게서 📎 근거가 붙은 답을 받았나요?** 아직 본인 자료가 아니어도, 출처를 밝히며 답하는 구조를 봤다면 이 챕터의 목적은 달성입니다.
- **"이 자리에 내 자료가 들어가면?"이 그려지나요?** 지금 시드 자리에 나리님의 강의·상담이 들어가면, 같은 답이 **본인 목소리**가 됩니다.

:::
