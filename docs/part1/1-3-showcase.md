# 1-3. 자동화, 어디까지?

::: tip Step 3 / 9 · Part 1
이 챕터에서 — 명령어를 익혔으니, **클로드 코드로 어디까지 자동화되는지** 강사의 실제 에이전트로 봅니다. "나도 이 정도 되겠구나"를 눈으로 확인하고 다음 챕터에서 직접 만들어요.
:::

## [보기] 강사의 콘텐츠 에이전트 — 지시 한 줄로

강사는 **딜 아카이브**(상업용 부동산 거래를 카드뉴스로 정리하는 채널)를 클로드 코드 에이전트로 운영해요. 강사가 시키는 건 사실상 한 줄이에요.

> "이번 주 딜 찾아서, 사실 확인하고, 카드뉴스로 만들어줘."

그러면 에이전트가 **세 가지 기능**을 이어서 해냅니다. 실제로 돌아간 과정을 그대로 보여드릴게요.

### ① 기사 리서치 기능 — 딜을 찾아온다

강사가 한 건 `run_crawl.sh` 한 줄 실행뿐이에요. 그러면 에이전트가 뉴스를 크롤해 상업용 부동산 거래 후보를 모으고, 자동으로 걸러 **후보 풀**을 갱신합니다.

```
크롤 정상 완료 (2026-07-17 13:02)
- gnews 리드 85건 수집 → 신규 후보 4건 추가 · 탈락 26건
- 후보 풀 총 23건 유지 (news_candidates.json)
```

새로 찾아온 딜은 위치·보도가와 함께 정리돼 나와요.

| 딜 | 보도가 | 위치 |
|---|---|---|
| 그랜드 성수 (준공 전 매각) | 미정 | 성동구 성수동 |
| JB금융 여의도 사옥 (매각 검토) | 1,000억 | 중구 서소문동 |
| 오렌지센터 (5월 오피스 거래) | 6,466억 | 중구 순화동 |

단순 수집이 아니에요. **중복·무관한 건 제외키워드(분양·재개발·아파트)로 걸러내고**, 쓸 만한 후보만 남깁니다. 심지어 *"이건 '분양' 키워드에 걸려 탈락했지만, 실제론 유효한 딜일 수 있어요"*처럼 **걸러낸 것까지 되짚어** 알려줘요.

### ② 제안 기능 — 뭘 만들지 골라준다

모은 후보 23건을 훑어 **카드화 가치**로 평가하고, 등급을 매겨 추천해요. 이미 만든 6건은 빼고요.

```
A급 (단일딜·한국·최신·확정 가능)
- 넷마블 사옥 6,976억 (신사옥, 넷마블→NH투자증권) — 최대 규모·화제성
- JB금융 여의도 사옥 1,000억 (서소문 신사옥 이전) — 깔끔한 단일딜
- 태영건설 독산동 노보텔 부지 (부채감축 매각)

B급 (딜은 유효하나 확인 필요)
- ABL생명 여의도 사옥 4,000억 → 우리금융
- 두산타워 매입 추진 (이지스-NH)

제외 권장
- 시장통계 기사(단일 딜 아님) · 해외 건(오스틴 오피스)
```

단순히 목록만 주는 게 아니라 **"이건 최대 규모라 화제성이 커요", "이건 아직 확인이 필요해요"**처럼 이유를 달아 골라줘요. 그리고 여기서 놀라운 지점 — 추천하기 전에 **사실을 스스로 교차검증**합니다. (정규식으로 뽑은 매도·매수 주체엔 파싱 오류가 섞일 수 있어, 선정 전 기사 원문으로 확정해요.)

::: tip 에이전트가 잡아낸 오류 (팩트체크)
- **"보도가 6,862억? 원문 확인하니 875억이네요."** — 크롤 단계 파싱 오류를 기사 원문으로 잡아냄
- **"JB금융은 아직 매각 '검토' 단계 — 확정가·매수자 없음."** — 확정 딜이 아닌 걸 구분해 "매각 추진 중"으로만 표기 제안
- 출처가 다르면 **어느 매체가 어느 수치의 근거인지** 나눠서 표기 (땅집고=875억 / 서울경제=매입가)
:::

## [개념] 방금 본 자동화의 정체

강사의 에이전트가 마법을 부린 게 아니에요. 핵심은 두 가지예요.

- **리서치 + 판단 + 제작을 지시 한 줄로** — 사람이 단계마다 붙어 있지 않아도 알아서 진행
- **정직함이 내장돼 있음** — 근거를 대고, 틀린 걸 스스로 잡고, 없는 건 "없다"고 함

::: tip 여기서 느끼셨으면 하는 것
"나도 이 정도 수준으로 자동화가 가능하겠구나."

맞아요. 그리고 오늘 나리님이 만드는 커리어 코치도 **똑같은 규율**로 일해요 — 리서치하고, 사실을 확인하고, 출처를 밝히고, 없는 건 지어내지 않습니다. 도메인만 부동산 콘텐츠에서 **커리어 코칭**으로 바뀔 뿐이에요.
:::

## [체크포인트] 자동화의 그림이 그려지나요

- **에이전트가 리서치→제안→제작을 한 줄로 해내는 걸** 봤나요?
- **에이전트가 사실을 스스로 확인하고, 없는 건 지어내지 않는 것**을 봤나요? (이게 오늘 만들 코치의 핵심이에요)
