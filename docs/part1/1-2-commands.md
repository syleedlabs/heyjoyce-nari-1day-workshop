# 1-2. 명령어 & 네비게이션

::: tip Step 2 / 9 · Part 1
이 챕터에서 — 클로드 코드를 움직이는 **두 가지 방법**(대화 / 슬래시 명령어)과, 자주 쓰는 명령어·입력 모드·단축키를 익혀 손에 붙입니다.
:::

## [개념] 두 가지로 시킬 수 있어요

클로드 코드에 일을 시키는 방법은 두 가지예요.

| 방법 | 어떻게 | 언제 |
|---|---|---|
| **그냥 대화** | 평소 말로 "이 폴더 자료 정리해줘" | 대부분의 작업 — 자유롭게 부탁·수정 |
| **슬래시 명령어** | `/`로 시작하는 미리 만든 명령 | 자주 쓰는 기능을 한 단어로 |

대부분은 **대화**로 하고, 아래 슬래시 명령들은 필요할 때 꺼내 쓰면 돼요. 외울 필요 없어요 — 전체는 `/help`로 언제든 볼 수 있어요.

## [개념] 자주 쓰는 슬래시 명령어

**세션 관리 (제일 많이 씀)**
| 명령 | 하는 일 |
|---|---|
| `/clear` | 대화를 초기화 (파일은 그대로 유지) |
| `/compact` | 대화를 압축 — 토큰 절약 + 앞부분 까먹기 방지 |
| `/resume` | 껐다 켜도 이전 대화를 복원 |

**모델 · 사용량**
| 명령 | 하는 일 |
|---|---|
| `/model` | Sonnet(빠름) ↔ Opus(똑똑함) 전환 |
| `/cost` | 이번 세션의 토큰·예상 비용 |
| `/usage` | 구독 플랜 사용량·한도 확인 |

**컨텍스트 · 파일**
| 명령 | 하는 일 |
|---|---|
| `/context` | 지금 인식 중인 파일 목록 보기 |
| `/add-dir` | 참고할 폴더 추가 |
| `/init` | 새 프로젝트에 `CLAUDE.md` 자동 생성 |
| `/memory` | 프로젝트 기억 확인·수정 |

**에이전트 · 권한 · 기타**
| 명령 | 하는 일 |
|---|---|
| `/agents` | 에이전트(코치) 생성·관리 |
| `/permissions` | 도구 사용 권한 설정 |
| `/help` · `/config` · `/doctor` | 전체 명령 보기 · 설정 · 문제 진단 |
| `/quit` · `/exit` | 클로드 코드 종료 |

## [개념] 입력 모드 — `Shift+Tab`으로 순환

클로드 코드는 "**어디까지 자동으로 승인할지**"를 모드로 정해요. `Shift+Tab`을 누르면 순환합니다.

| 모드 | 뜻 | 언제 |
|---|---|---|
| **accept edits on** | 파일 편집은 자동 승인, 셸 명령은 승인 요청 | 익숙해진 신뢰 작업 |
| **plan mode on** | 실행 전 **계획을 먼저 보여주고** 승인하면 실행 | 복잡한 작업을 단계별 확인 |
| **auto mode on** | 편집·명령 모두 자동 승인 | 완전 자동화 |

기본은 셋 다 꺼짐(수동 승인). 처음엔 기본으로, 익숙해지면 하나씩 켜세요.

## [개념] 키보드 단축키

| 단축키 | 기능 |
|---|---|
| `Esc` | 클로드 응답 중단 |
| `Esc` ×2 | 되감기 (대화만 또는 파일 변경까지) |
| `Shift+Tab` | 입력 모드 순환 |
| `Shift+Enter` | 줄바꿈 (여러 줄 입력) |
| `Ctrl+V` | 이미지 붙여넣기 |

## [개념] "더 생각해줘" 키워드

어려운 판단엔 `think` 계열 키워드를 붙이면 코치가 더 깊이 생각해요. (그만큼 시간·비용은 늘어요.)

| 키워드 | 언제 |
|---|---|
| `think` | 가벼운 비교·정리 ("이 두 문구 중 뭐가 더 끌려?") |
| `think hard` | 여러 요소를 동시에 고려하는 결정 |
| `ultrathink` | 되돌리기 어려운 중요한 결정 |

## [실습] 직접 눌러보기

거창한 것 말고, 조작 몇 개만 손에 붙여봐요.

1. **`Shift+Tab`**을 몇 번 눌러 입력 모드가 바뀌는 걸 보세요 (기본 → accept → plan → auto → 기본).
2. **plan mode**에서 *"이 폴더를 분석하고, 여기서 어떤 작업이 가능한지 제안해줘"* 라고 해보세요 — 코치가 실행 전에 **계획부터** 보여줘요.
3. **`/compact`**로 대화를 압축하고, **`/cost`**로 사용량을 확인해 보세요.

::: tip 완벽히 외울 필요 없어요
지금 다 못 외워도 돼요. 막히면 `/help`, 헷갈리면 그냥 대화로 물어보면 됩니다. 자주 쓰다 보면 손에 붙어요.
:::

## [체크포인트] 여기까지 잡으셨다면

- **대화 / 슬래시** 두 방법이 있다는 것
- 세션 관리 3종 — `/clear` · `/compact` · `/resume`
- **`Shift+Tab`** 모드 순환 · **`Esc`** 중단
- 막히면 `/help`
