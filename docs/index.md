## 워크북 사용법

이 워크북은 **직접 따라하며 배우는 실습 중심** 자료입니다.

### 구조
- **[개념]** — 읽고 이해하는 파트
- **[실습]** — 직접 수행하는 파트 (슬래시 커맨드 호출)
- **[체크포인트]** — 진행 상황 확인 + 주요 마일스톤

### 소요시간
약 **4시간** (Prologue 5분 + Part 1 기초·맛보기 + Part 2 개념·빌드)

### 사전 준비물
- 노트북, 인터넷 연결
- 클로드 코드 설치 (Pro/Max 구독) — 이미 설치돼 있어도 OK, 1-2에서 함께 재점검합니다
- **본인 커리어 코칭 자료 3~5개** — 강의 PPT 2~3개, 영상 1~2개(자막 있으면 더 좋아요), 대표 상담 사례 메모. 전량 아니어도 됩니다, 대표 자료면 충분해요.

### 학습 모드
- 프롬프트가 완벽하지 않아도 괜찮습니다. 코치는 **클로드 코드와의 대화로 언제든 수정**할 수 있어요.
- 모르는 부분 있으면 멈추고 물어보세요.
- **상담 노트의 실제 내담자 정보는 익명화**해서 넣습니다. 개인정보 원칙은 [2-4](/part2/2-4-build)에서 자세히.
