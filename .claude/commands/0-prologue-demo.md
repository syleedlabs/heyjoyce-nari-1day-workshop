워크숍 시작 시 호스트(강사)가 학습자에게 보여줄 **그라운딩 답변 샘플**을 출력합니다. 강사가 본인 코치로 라이브 데모를 못 보여줄 때의 백업용.

`docs/prologue.md`의 "잠깐, 실물을 먼저 보시죠" 블록과 **동일한 형식**으로 출력하세요. 핵심은 세 가지가 다 보이는 것:

1. **결론이 강사의 관점** (범용 정답이 아니라 강사 프레임)
2. **출처 표기** — `📎 근거: <자료명> (슬라이드/영상 구간)`
3. **자료 밖 라벨** — `⚠️ 여기서부턴 제 자료 밖 일반론입니다`

## 출력 형식

```
[코치] 답변

<결론 — 강사 화법으로, 강사의 프레임을 담아>

📎 근거: <강사 자료명> (슬라이드 N / 영상 MM:SS)

⚠️ 여기서부턴 제 자료 밖 일반론입니다: <자료 밖 내용>

💬 이어서 묻고 싶은 것: <다음 질문 1개>
```

## 기본 샘플 (강사 자료가 없을 때)

프롤로그 본문의 예시("AI 때문에 제 직무가 사라질까 봐…")를 그대로 출력해도 됩니다. 기본 출처는 `「강사의 커리어 강의」`로 표기.

**권장**: 강사(이수연)는 본인의 강의 자료를 `knowledge/`에 넣어두고 그 실제 자료명·슬라이드 번호로 답하세요. 데모의 📎 근거가 **지어낸 게 아니라 진짜 출처**여야, 그라운딩 원칙을 몸소 보여주는 데모가 됩니다.
