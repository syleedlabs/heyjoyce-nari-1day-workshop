2-2에서 배운 PRD를 코치에 맞춰 확정한다. 이후 인제스트·튜닝의 **기준선**이 되는 한 장짜리 명세다.

## 1. 재료 로드
`.claude/agents/코치.md`(페르소나 초안, 1-3에서 작성)와 1-3 인터뷰 답변을 읽는다.

## 2. 코치 PRD 초안 작성
아래 표를 채워 화면에 보여준다. 비어 있으면 나리님에게 한 항목씩 짧게 묻는다.

| 항목 | 내용 |
|---|---|
| **대상(누구)** | 주 코치이 — 예: "직무 전환 고민 3~10년차" |
| **다루는 고민(무엇)** | 커리어 설계·직무 전환 코칭 — 예: "이직·직무전환·커리어 설계" |
| **근거(무엇으로)** | 지식 베이스 `knowledge/` (본인 강의·영상·상담) |
| **톤** | 나리님의 화법 (인터뷰 2번) |
| **품질 기준** | ⚠️ **그라운딩 철칙 필수**: 답변마다 `📎 근거`, 코퍼스 밖은 "일반론" 라벨, 지어내기 금지 |
| **안전 정책** | 초안까지만(자동 발송 X) · 민감정보 익명화 · 의료·법률·재무는 전문가 권유 |

## 3. 확인
- **품질 기준에 그라운딩 철칙이 들어갔는지** 반드시 확인시킨다. 이게 빠지면 코치가 지어낼 수 있다.
- 나리님이 OK 하면 PRD 요지를 페르소나 §2(도메인)·§4(그라운딩)·§7(안전)에 반영한다.
