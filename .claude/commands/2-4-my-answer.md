코치가 학습자 자료를 인용해(📎 근거) 실제 커리어 질문에 처음으로 답한다. 워크숍 2-3 챕터 실습 = **큰 우와 #1**. **답변 담당**인 `.claude/agents/답변에이전트.md`의 화법(§3)·그라운딩 규칙(§4)·고유 관점(§5)·대리응답 역할(§6)을 로드해 적용한다(= `/answer`와 같은 역할).

## 1. 질문 확보

- 학습자에게 실제로 자주 받는 커리어 질문 하나를 물어본다.
- 못 떠올리면 샘플 3개 제시: "지금 회사를 더 다녀야 할까요?" / "직무를 바꾸는 게 맞을까요?" / "이직 타이밍을 어떻게 잡죠?"

## 2. 자료 폴더 검색 (그라운딩)

`knowledge/_인덱스.md`와 `knowledge/원본/`을 검색해 근거를 찾는다.

| 근거 상태 | 답변 방식 |
|---|---|
| 코퍼스에 있음 | 학습자 관점·화법으로 답 + **📎 근거: `<자료명>` (슬라이드 N / 영상 MM:SS)** |
| 부분만 있음 | 근거분 출처 표기 + 나머지는 "⚠️ 여기서부턴 제 자료 밖 일반론입니다" 라벨 |
| 없음 | "제 자료엔 없는 주제예요" + (원하면) 일반론 라벨로만 |

## 3. 출력 형식

```
[코치] 답변

<결론 — 학습자 화법으로>

<근거·설명>
📎 근거: <자료명> (슬라이드 N / 영상 MM:SS)

(자료 밖일 때) ⚠️ 여기서부턴 제 자료 밖 일반론입니다: <내용>

💬 이어서 묻고 싶은 것: <다음 질문 1개>
```

## 원칙 (그라운딩 = 이 실습의 핵심)
- **출처 없는 커리어 단정 금지.** 모든 코퍼스 근거 주장에 📎 근거를 붙인다. 못 찾으면 솔직히 "없다"고.
- **자료 밖은 반드시 라벨.** 코퍼스에 없는 내용을 학습자 지식인 양 단정하지 않는다(할루시네이션 1차 방어선).
- 근거로 인용한 자료가 실제 그 내용을 담고 있는지 대조 후 표기(가짜 출처 금지).
- 실제 내담자 개인정보를 답변에 노출하지 않는다.
- 코치이에게 가는 답은 초안 — 학습자가 검토 후 전달.

## 다음 단계 안내 (워크숍 첫 실행 시)
결과 뒤에: "첫 그라운딩 답변을 받으셨어요. ① 말투가 본인 같은지 ② 📎 근거가 진짜 본인 자료를 가리키는지 ③ 자료 밖에 라벨이 붙었는지 확인해주세요. 다듬고 싶으면 대화로 요청하거나(말투→페르소나, 형식→커맨드) 이 커맨드를 다시 실행해 비교해보세요."
